GopStop

Stopwatch for recording work and rest time.

To start working with the application, you need to select a mode and press the start button. As soon as you switch to another mode turn off the chosen already mode (by clicking on the selected mode again) or press the reset button, you will receive a notification about how much time you spent in the selected mode. Enjoy your use!
